INTRODUCTION TO STATISTICAL LEARNING

This is an introductory-level course in supervised learning, with a focus on regression and classification methods. This is not a math-heavy class, so I try to describe the methods without heavy reliance on formulas and complex mathematics. Computing is done in R. There are labs devoted to R with more detailed sessions that implement the techniques in each lesson.
