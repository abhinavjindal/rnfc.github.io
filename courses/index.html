End User Modeling

Target Audience

HBA students who would like to develop advanced modeling and spreadsheet skills, and learn VBA (Visual Basic Applications) programming for Excel.

Purpose/Objectives

The course will freshen and sharpen Excel and management science modeling concepts and skills, and introduce programming with VBA and provide a variety of applications (e.g., simulation, optimization) from various fields (e.g., finance, operations, service).

Overview

  • Excel and management science modeling brush-up (this will be done as necessary throughout the course)
  • Overview of VBA (e.g., objects, ranges, macros, input, output, VBA editor)
  • Graphical interface components (e.g., forms, controls)
  • Programming structures (e.g., variables, assignments, loops, controlling program flow, functions, arrays), and debugging and error handling
  • Special topics/hints (e.g., dynamic charts, getting data from Internet, pivot tables, data validation tool, data analysis)
  • Topics for management science applications (e.g., solver controls with VBA, simulation with VBA, automation of data analysis tool pack usage with VBA)
  • Application/Decision Support System (DSS) development process
  • Examples of applications/DSS

Materials/Equipment

An up-to-date working copy of Microsoft Excel (Windows Excel is highly recommended even though now Mac Office 2011 has (some) VBA support). In addition to Excel, a few Excel add- ins will be used in this course, e.g., solver, data analysis, VBA help.
